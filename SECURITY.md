# Security Policy

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues,
discussions, or pull requests.

Report them privately via GitHub's private vulnerability reporting:
<https://github.com/imprun/windforce-core/security/advisories/new>

Include reproduction steps, the affected version or commit, and the impact you
expect. You will receive a response as soon as possible, normally within a few
days.

## Supported Versions

Only the latest release and the current `main` branch receive security fixes.
